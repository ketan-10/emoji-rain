## Setup

`git clone https://github.com/ketan-10/emoji-rain.git`

`cd emoji-rain`

`yarn` : Install dependencies

`yarn dev` : Serve application


**Dependencies Used :**

- @material-ui/core : styling
- react-spring : animation
